Help

Drag & Drop Images

Drag any image onto the canvas. The top half replaces the icon, the bottom half replaces the frame.

Inline Icons in Descriptions

Type <imgkeyword> in the description to embed a stat icon inline with text.

Browse available icons in the Icons collapsible section — click any icon to insert it at your cursor.

Icon Size Modifiers

Adjust the size of individual inline icons:

  • <imgprismatic_vamp+4> — 4px larger than font size
  • <imgscaleAP-3> — 3px smaller than font size

By default icons render at font size − 2px.

Color Tags

Wrap text in color tags for colored inline text. Names come from the Colors section.

  • <gold>50 AD</gold>
  • <magicDamage>80</magicDamage>
  • <prismatic>Prismatic text</prismatic>

For arbitrary colors: <fontcolor='#FF4500'>text</fontcolor>

Insert Inline References (+ Button)

Click the + button on any augment, champion, or item to insert it as a small colored icon reference in your description at the cursor position.

Custom Augments

Click Save Augment to store your current design (title, description, icon, frame, fonts). Find saved augments under the Custom tab.

Presets

Save layout configurations (positioning, fonts, spacing) as named presets with the save button next to the Presets dropdown. Useful if you switch between different card styles.

Colors Section

Add, rename, or change any color in the color table. These colors are what the <colorName> tags reference. Changes persist in your browser.

Other Tips

  • The rules color tag also makes text italic and faded
  • Selecting a new augment auto-clears any custom frame override
  • Right-click the canvas image to save it
  • All settings are saved in your browser — nothing is uploaded
